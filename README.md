# Login / Sign Up Page

This is a simple **Login / Sign Up webpage** created using **HTML**, **CSS**, and **JavaScript**.

--

## Features
- Sign Up with username and password (stored in browser localStorage)
- Login with the saved credentials
- Alerts for success or failure
- Beginner-friendly, clean, and simple design

---

## Files in this project
- `index.html` → The main login/signup page
- `style.css` → CSS for styling
- `script.js` → JavaScript for login/signup logic

---

## How to use
1. Open `index.html` in your browser.
2. Sign up with a new username and password.
3. Login with the same credentials.
4. Get alerts like:
   - ✅ *Login successful!*  
   - ❌ *Invalid credentials!*  

---

## Notes
- This project is for **learning only**.  
- Passwords are saved in localStorage (not safe for real-world apps).  
- You can improve it by adding proper validation or connecting to a backend server.
